## 🎨 Screenshot Best Practices

### Image Specifications
- **Format:** PNG (for quality) or JPG (for smaller size)
- **Dimensions:** 1200x600px to 1200x700px (recommended)
- **Quality:** High resolution (72+ DPI)
- **File Size:** < 500KB (compress if needed)
- **Background:** Include full page with context

### Capture Tips
1. **Use Full Browser Window** - Show complete interface
2. **Include Navigation** - Show navbar and footer
3. **Highlight Features** - Make key elements visible
4. **Use Real Data** - Show actual content, not placeholders
5. **Clean UI** - No error messages or debug info
6. **Consistent Theme** - All screenshots should match styling

### Tools for Screenshots
- **Windows:** Snipping Tool, ShareX, or Greenshot
- **Mac:** Screenshot tool or Skitch
- **Online:** Figma, Canva for mockups
- **Browser:** Built-in DevTools screenshot

### Image Optimization
```bash
# Using ImageMagick
convert input.png -resize 1200x700 -quality 85 output.png

# Using TinyPNG
# Visit: https://tinypng.com/

# Using FFmpeg
ffmpeg -i input.png -vf scale=1200:700 output.png
```
